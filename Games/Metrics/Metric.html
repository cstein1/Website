by Charlie Stein cnstein22 `AT` gmail `DOT` com

A unit circle is defined by the set of points which are a distance of 1 away from the origin. Distance is a very arbitrary concept! Clicking on the buttons will place the given distance function (metric) in the text box and display the unit circle in that metric! Feel free to input your own metric and click the "Apply 2D Metric" button. The textbox compiles to Javascript and only accepts variables x, y, and any mathematical function. The circle is computed using quad trees. To see how I calculate the circle coordinates, hit the "Toggle Computation" buttons.
